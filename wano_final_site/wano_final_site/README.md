# République of Wano (Static Site)

Site statique HTML/CSS/JS prêt pour Netlify.

## Fichiers
- `index.html` — Page d’accès (login + dashboard). Le lien **Latest Release** pointe vers `tracks_detail.html`.
- `tracks_detail.html` — Liste détaillée des pistes (recherche, filtres, player, raccourcis).
- `tracks.html` — Version mosaïque des pistes.

## Déploiement
- **Drag & drop** le dossier entier dans Netlify **(les 3 fichiers doivent être à la racine)**.
- Ou mets-les à la racine d’un repo GitHub et utilise *Import from Git*.

## Audio
Dans les tableaux `TRACKS`, remplace les `src:''` par tes URLs (ou chemins relatifs) de fichiers audio.
